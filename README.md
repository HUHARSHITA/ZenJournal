# ZenJournal
ZenJournal is a minimal, aesthetic, and responsive daily journal desktop application built with Python. It features a "Clean Slate" philosophy‚Äîgiving you a fresh page every day while securely preserving your past memories in a beautifully themed vault.
Features
1. One Entry Per Day: Encourages consistent reflection with a clean slate every morning.Dynamic "Vibe"
2. Themes: Cycle through curated palettes including Lavender Dream, Sunny Sunstone, Matcha Moss, and Sunset Rose.Persistent
3. Memory: Your entries and theme preferences are saved locally in a secure JSON vault in your user directory.
4. Responsive Design: Fully adaptive UI that looks elegant at any window size.
5. Liquid Smooth UI: Powered by CustomTkinter for rounded corners, smooth transitions, and hover scaling animations.
6. Custom Branding: Integrated logo support for a personalized touch.
Note: Your journal data will be safely stored in %USERPROFILE%\ZenJournal_Data to ensure it isn't deleted when the app closes.
